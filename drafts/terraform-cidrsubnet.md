# Terraform `cidrsubnet`

Terraform 에 `cidrsubnet` 이라는 함수가 있는데, 설명을 읽어도 이해가 안되서, 
삽질을 좀 하며 이해한 내용을 정리합니다.

더불어 `ipcalc` 라는 완소 유틸리티도 사용해보세요.

```
cidrsubnet("10.0.0.0/20", <newbit>, <newnum>)
```
